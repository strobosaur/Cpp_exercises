The final exercise of this C++ course was writing a program solving the following case:

A group of friends wants to go on vacation together. 
Whenever they rent a car, order dinner, pay for lodging etc. one of them foots the bill.
Which of his/her friends that take part in a particular acitivity and transaction varies.
At the end of the trip, they need to work out their individual debts, from a list of transactions.
The program reads the transactions from the file "resa.txt" and works out the numbers.
